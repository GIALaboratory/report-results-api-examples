### GIA Report Results API Python Example


``` bash
git clone https://github.com/GIALaboratory/report-results-api-examples.git
cd report-results-api-examples/python

pip install -r requirements.txt

# Configure environmental variables
set REPORT_RESULTS_API_KEY=INSERT_YOUR_API_KEY_HERE
set REPORT_RESULTS_API_ENDPOINT=INSERT_THE_API_ENDPOINT_HERE

python server.py
# Go to http://localhost:5000
```
